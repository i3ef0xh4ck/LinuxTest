python版本2.7

安装过程
1.安装python和配置Python2.7的环境变量
Windows下双击安装python-2.7.9.amd64.msi即可

2.安装pycrypto
双击安装pycrypto-2.6.win-amd64-py2.7.exe即可

3.安装ecdsa包
安装包下载地址：
解压ecdsa-0.11.tar.gz
安装命令如下
python setup.py build
python setup.py install

4.安装setuptools
安装包下载地址：
解压setuptools-20.9.0.tar.gz
安装命令如下（同安装paramiko包）
python setup.py build
python setup.py install

5.安装paramiko包

6.安装xlrd、xlwt
这两个包用于excel操作
